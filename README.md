# 🎬 Movie Collection App

A simple and interactive **Movie Collection Web Application** built using **HTML, CSS, and JavaScript**, integrated with the **OMDb API**.  
Users can log in, create an account, and search for movies with details like poster, year, and type.

---

## 🚀 Features

- 🔐 User Login System
- 📝 Create Account (Sign Up)
- 🎥 Search Movies using OMDb API
- 🖼️ Display Movie Posters & Details
- 📱 Responsive UI
- 🔄 Logout Functionality

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OMDb API
- LocalStorage (for new user accounts)
